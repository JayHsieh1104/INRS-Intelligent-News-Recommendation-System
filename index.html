System Architecture




Architecture

Analyzed articles are uploaded and stored on AWS postgreSQL database. The front-end web and the back-end server are both hosted on GCP (Google Cloud Platform).

Data

We parsed 300+ news articles from the RESTful APIs of 50+ well-known online news publishers, such as CNN, Fox News, CBS News, Los Angeles Times, and so on.

Approach

A python script will fetch news data from the news APIs of the pre-selected publishers. With natural language processing and machine learning methods, the script performs semantic analysis on the fetched data and then inserts the analyzed data into the database. The front-end web will present articles based on users' selected category. The back-end server is responsible for querying the database and providing the front-end with queried news data.

Model (Unfinished)

The three models developed for ChestX.ai employ state-of-the-art DenseNet-121 convolutional neural network (CNN) model architecture. Additionally, heatmaps for localized important regions are generated using Gradient-weighted Class Activation Mapping (Grad-CAM).
